# PitStop Parts

An e-commerce web app for a Kenyan auto parts store. Browse parts, search by vehicle or part number, add to cart, and place orders for delivery. Staff can manage products and orders through an admin panel.

## Features
- Product catalogue with search, category, and condition filters
- Vehicle make/model search with dynamic dropdown
- Shopping cart and checkout with shipping details
- User registration and login
- Admin panel — add, edit, delete products and update order status

## Tech Stack
PHP · MySQL · JavaScript · HTML/CSS · Apache
